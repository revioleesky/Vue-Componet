### 3.可选配置说明

| 属性            | 说明                       | 类型   | 默认值                 |
|-----------------|----------------------------|--------|------------------------|
| scale           | 缩放比例（选填）           | String | 1                      |
| num             | 显示+num数字               | String | -                      |
| name            | 中文名                     | String | -                      |
| mode            | 选择显示圆形还是方形的头像 | String | 填入square显示方形头像 |
| fontColor       | 字体颜色                   | String | #fff                   |
| backgroundColor | 背景颜色                   | String | #3696F2                |
| faceUrl         | 图片地址                   | String | -                      |
